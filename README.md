
 Weather App 🌦️

A simple weather application built with **React + Vite** that fetches real-time weather data using an external API.

##  Features
- Search weather by city
- Displays temperature, humidity, and conditions
- Responsive UI with weather icons

##  Tech Stack
- React + Vite
- JavaScript (ES6+)
- CSS
- OpenWeather API

## 📂 Project Setup
Clone the repository and install dependencies:
```bash
git clone https://github.com/VulavalaMounika/Weather-App.git
cd Weather-App
npm install
npm run dev
